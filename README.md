# MyMoney

Личное приложение для учёта финансов под iOS: счета, операции, сводка и долги.
Работает офлайн, без авторизации, все данные хранятся локально на устройстве.

## Возможности

- **Счета** — наличные, карты, кредитные карты и т.д. Мультивалютность, начальный
  баланс, кредитный лимит. Общий баланс с пересчётом в базовую валюту по заданным
  курсам. Суммы скрываются глазком в заголовке «Общий баланс».
- **Операции** — приход, расход и перевод между счетами (в т.ч. в разных валютах).
  Категории, хештеги с подсказками (зависят от выбранной категории), погашение
  долгов прямо из операции, поиск и фильтры.
- **Сводка** — кольцевая диаграмма и разбивка расходов/доходов по категориям с
  вложенной детализацией по хештегам, за месяц или год.
- **Долги** — «я должен» и «мне должны», частичное и полное погашение.
- **Настройки** — управление категориями, базовая валюта, курсы валют.

Навигация — нижняя панель вкладок + горизонтальный свайп между экранами.

## Стек

- **SwiftUI** — интерфейс
- **SwiftData** — локальное хранилище
- **Swift Charts** — диаграммы в сводке
- Цель — **iOS 17+**

## Структура

```
MyMoney-swift/
  MyMoney-swift/
    Models/     — Account, Transaction, TransactionCategory, Debt, CurrencyRate
    Views/      — экраны и формы
    Helpers/    — расчёты баланса/сводки, валюты, вёрстка
    ContentView.swift — корневой экран с пейджером и таб-баром
docs/           — проектные заметки (зашифрованы через git-crypt)
secrets/        — зашифрованный ключ git-crypt
```

## Сборка и запуск

```bash
open MyMoney-swift/MyMoney-swift.xcodeproj
```

Собрать и запустить на симуляторе можно из Xcode (⌘R) или из консоли:

```bash
xcodebuild -project MyMoney-swift/MyMoney-swift.xcodeproj \
  -scheme MyMoney-swift \
  -destination 'platform=iOS Simulator,name=iPhone 17' build
```
